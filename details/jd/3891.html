  Créer sa boutique en ligne avec prestashop 
Organisme : KOMELYA
6 Rue de Porstrein - 
29200 BREST
Tél : 02 98 70 03 42 - 
Email : contact@komelya.fr 
Site web : http://www.komelya.fr 

Réf : 1300691F - Mise à jour : 20/08/2013
Contact pour la formation : Christelle HERVIEUX (Responsable Formation)



Objectifs

Acquérir une autonomie pour configurer et administrer un site web marchand professionnel



Condition d'admission
  • Une bonne connaissance de l'environnement informatique Macintosh ou PC, ainsi qu'une pratique régulière d'Internet.
  • Public visé
    Cette formation s'adresse avant tout aux PME, artisans, commerçants, auto-entrepreneur qui souhaitent créer une boutique en ligne.


Programme

1) Les fondamentaux
- Qu'est-ce qu'un site web ? les types de site ( forum, achats en ligne...)
- Le contenu d'un site web : images, vidéos, HTML, animations flash.
Les échanges entre navigateur et site web
- Le rôle du navigateur (Internet Explorer, Firefox,...) et du serveur web.
- Comment se comprendre entre les 2 (IP, DNS et protocole HTTP) ?
2) L'environnement technique d'un site web
Comprendre l'architecture d'un site web
Conception d'un site internet : les outils nécessaires
Travaux pratiques : configurer la base de données chez l'hébergeur
3) L'hébergement et le nom de domaine du site
4) A propos de Prestashop
Travaux pratiques :
- Installer Prestashop sur son serveur web en local (sur son ordinateur)
- Installer Prestashop via le logiciel FTP sur le serveur web professionnel et paramétrage base de données MySQL
5) Connaître le back-office de Prestashop
6) Réaliser la page d'accueil de la boutique et les pages produits
7) Personnaliser sa boutique
- Les outils nécessaires.
- Architecture et organisation des dossiers.
- Créer un nouveau thème à partir du modèle de base.
- Déplacer les blocs dans la page.
Travaux pratiques : Amélioration des pages web avec des fonctions avancées
8) Promotions et opérations marketing
9) Mode de paiement sécurisé
10) Référencer sa boutique



Durée de formation
  • 21 heures
Organisation pédagogique

Temps plein ;

Les formateurs assurent un enseignement professionnel qui alterne entre théorie et pratique. Après avoir suivi une première formation, tous nos stagiaires ont accès à notre communauté pour prolonger l'expérience formation et développer ses compétences bien au delà de la formation en groupe : supports de formation complémentaires, vidéos, tutoriels et être informé des dernières tendances sur le web et la communication.
